# Contract Settings

Configure the settings of your smart contract.

## Contract Metadata

Read, write, and update the metadata of a smart contract using the SDK,
such as the name, description and image.

### Read

<ThirdwebFeatureCodeSnippet
  featureName={"ContractMetadata"}
  languageFunctionMapping={{
    javascript: "get",
    react: "useContractMetadata",
  }}
/>

### Write (Overwrite)

:::warning
This will **overwrite** the existing metadata.
:::

<ThirdwebFeatureCodeSnippet
  featureName={"ContractMetadata"}
  languageFunctionMapping={{
    javascript: "set",
    react: "useUpdateMetadata",
  }}
/>

### Update

<ThirdwebFeatureCodeSnippet
  featureName={"ContractMetadata"}
  languageFunctionMapping={{
    javascript: "update",
    react: "useUpdateMetadata",
  }}
/>
