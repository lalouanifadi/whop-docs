# ERC20SignatureMintVote

The `ERC20SignatureMintVote` smart contract adds [signature-based minting](/contractkit/erc20signaturemintable)
to the [ERC20Vote](/contractkit/base-contracts/erc-20/erc20vote) base contract.
It also implements [ERC20Votes](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/contractkit/ERC20Votes.sol), which provides voting and delegation functionality, along with delegation by signature.

## Implementing the Contract Extension

Import the contract extension and make your contract inherit it.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/base/ERC20SignatureMintVote.sol";

contract MyToken is ERC20SignatureMintVote {
	  constructor(
        string memory _name,
        string memory _symbol,
        address _primarySaleRecipient
    )
        ERC20SignatureMintVote(
            _name,
            _symbol,
            _primarySaleRecipient
        )
    {}
}
```
