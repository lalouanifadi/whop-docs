# Features

        Features are individual pieces of functionality you can add to your smart contracts.

Each feature you implement in your smart contract unlocks corresponding functionality for you to utilize in the SDK.

There are two kinds of features:

1. **Contracts**: Fully implemented features you can import and use.
2. **Interfaces**: Standards you must implement the required functionality to utilize.

We provide several features for adding functionality to your [ERC721](/contractkit/erc721), [ERC1155](/contractkit/erc1155), and
[ERC20](/contractkit/erc20), as well as general extension for common use cases.

## Detecting Features

You can detect all of the features your contract implements using the CLI:

```bash
npx thirdweb detect
```

## List of Features

Below is a list of all the features available.

### ERC721

| Name                                                              | Description                                                                    |
| ----------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| [ERC721](/contractkit/erc721)                                     | Implementation of the [ERC721A](https://www.erc721a.org/) standard.            |
| [ERC721Mintable](/contractkit/erc721mintable)                     | Mint new NFTs to your ERC721 contract.                                         |
| [ERC721BatchMintable](/contractkit/erc721batchmintable)           | Mint multiple NFTs in a single transaction.                                    |
| [ERC721Supply](/contractkit/erc721supply)                         | View the total number of NFTs minted.                                          |
| [ERC721Enumerable](/contractkit/erc721enumerable)                 | View the current wallet address that owns any NFT.                             |
| [ERC721LazyMintable](/contractkit/erc721lazymintable)             | Batch Lazy Mint NFTs.                                                          |
| [ERC721Claimable](/contractkit/erc721claimable)                   | Allow users to claim a lazy-minted NFT under conditions you set.               |
| [ERC721ClaimableWithConditions](/contractkit/erc721batchmintable) | Set up claim-conditions that define how your users can claim lazy minted NFTs. |
| [ERC721Revealable](/contractkit/erc721revealable)                 | Add delayed reveal to your lazy-minted NFTs.                                   |
| [ERC721Burnable](/contractkit/erc721burnable)                     | Allow wallets to burn NFTs.                                                    |
| [ERC721SignatureMint](/contractkit/erc721signaturemintable)       | Add signature-based minting capability.                                        |

### ERC1155

| Name                                                                | Description                                                                                 |
| ------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| [ERC1155](/contractkit/erc1155)                                     | Implementation of the [ERC1155](/contractkit/base-contracts/erc-1155/erc1155base) standard. |
| [ERC1155Mintable](/contractkit/erc1155mintable)                     | Add the capability to mint new NFTs to your ERC721 contract.                                |
| [ERC1155BatchMintable](/contractkit/erc1155batchmintable)           | Mint multiple NFTs in a single transaction.                                                 |
| [ERC1155Enumerable](/contractkit/erc1155enumerable)                 | View the total number of minted NFTs and the amount of NFTs owned by a wallet.              |
| [ERC1155LazyMintable](/contractkit/erc1155lazymintable)             | Batch Lazy Mint NFTs.                                                                       |
| [ERC1155Claimable](/contractkit/erc1155claimable)                   | Allow users to claim a lazy-minted NFT under conditions you set.                            |
| [ERC1155ClaimableWithConditions](/contractkit/erc1155batchmintable) | Set up claim-conditions that define how your users can claim lazy minted NFTs.              |
| [ERC1155Revealable](/contractkit/erc1155revealable)                 | Add delayed reveal to your lazy-minted NFTs.                                                |
| [ERC1155Burnable](/contractkit/erc1155burnable)                     | Allow wallets to burn NFTs.                                                                 |
| [ERC1155SignatureMint](/contractkit/erc1155signaturemintable)       | Add signature-based minting capability.                                                     |

### ERC20

| Name                                                            | Description                                                                           |
| --------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| [ERC20](/contractkit/erc20)                                     | Implementation of the [ERC20](/contractkit/base-contracts/erc-20/erc20base) standard. |
| [ERC20Mintable](/contractkit/erc20mintable)                     | Add the capability to mint new tokens to your ERC20 contract.                         |
| [ERC20BatchMintable](/contractkit/erc20batchmintable)           | Mint tokens to multiple addresses in a single transaction.                            |
| [ERC20ClaimableWithConditions](/contractkit/erc20batchmintable) | Set up claim-conditions that define how your users can claim tokens.                  |
| [ERC20Burnable](/contractkit/erc20burnable)                     | Allow wallets to burn tokens.                                                         |
| [ERC20SignatureMint](/contractkit/erc20signaturemintable)       | Add signature-based minting capability.                                               |

### General

| Name                                              | Description                                                                        |
| ------------------------------------------------- | ---------------------------------------------------------------------------------- |
| [ContractMetadata](/contractkit/contractmetadata) | Add metadata such as name, description, symbol, and image to your smart contract.  |
| [Ownable](/contractkit/ownable)                   | Set the wallet address that owns the smart contract.                               |
| [Permissions](/contractkit/permissions)           | Add roles and restrict which roles can call functions.                             |
| [PlatformFee](/contractkit/platformfee)           | Charge a percentage fee wherever there is a transfer of currency in your contract. |
| [PrimarySale](/contractkit/primarysale)           | Set a recipient for any sale value you collect in your smart contract.             |
| [Royalty](/contractkit/royalty)                   | Take a percentage fee on secondary sales of tokens.                                |
