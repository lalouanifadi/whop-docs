Whop is an all in one platform for merchants to sell access to digital products.
Thousands of software developers, content creators, traders, and savvy internet folks use Whop to sell software, web apps, Discord servers, courses, files and many other digital things.
Integrating with the Whop API gives you complete control over the experiences you offer to customers, the memberships your users own, how you manage your payments, and much more!

To begin integrating with Whop, you'll first need to make sure you've created a company on the [Whop Dashboard](https://dash.whop.com).
![Image](./assets/create-company.png)

<p style={{ marginBottom: 6 }}>
  Once you have a company set up, the next steps depend on your end goal:
</p>

- If you want to quickly setup integrations with Whop to add some extra functionality to your project, we recommend checking out our [Quick Integration guide](/getting-started/business-app)
- If you're looking for a little more granularity, our [Full Integration guide](/getting-started/custom-experience) will lead you through every step of setting up a project using the Whop API
- If you just want to dive in to some code, our [NextJS Template App](/nextjs) is an easy solution. Built on the lastest version of NextJS, this template app is already optimized for integrations with Whop, letting you focus on what's important to your customers
