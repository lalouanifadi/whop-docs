# Connect Wallet Button

The Connect Wallet button allows the user to connect to your application using their
[MetaMask](/react/react.usemetamask),
[WalletConnect](/react/react.usewalletconnect) or [Coinbase](/react/react.usecoinbasewallet) wallet.

After the user has connected, the user can:

- View the connected wallet address
- View their balance of the **native token** _(such as ETH)_
- Switch network, disconnect wallet, and copy their wallet address
